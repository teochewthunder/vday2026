# Valentine's Day Ribbon

## Elements
- Path tag with background using `valGradient`.
- Path tag with background using `valGradient2`.
- Path tag with background using `valGradient2`.

## Gradients
- `valGradient`: Linear; from red to pink
  - animate `stop-color` attribute to go from red to pink to red.
- `valGradient2`: Linear; from pink to red
  - animate `stop-color` attribute to go from pink to red to pink.
